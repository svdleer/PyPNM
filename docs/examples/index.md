# PyPNM Examples

This directory contains runnable examples showing how to use the PyPNM project in four main contexts.

```text
examples/
├── common          # Shared example resources
├── fast_api        # FastAPI endpoint examples
├── python          # Python API examples
├── service         # Service layer examples
├── settings        # Configuration and settings examples
└── snmp            # SNMP API examples
```

| Guide | Description |
| ----- | ----------- |
| [FastAPI PNM](cli-pnm-fastapi.md)         | Examples demonstrating the FastAPI endpoints for PNM Signal Analysis and reporting.               |
| [FastAPI SNMP](cli-snmp-fastapi.md)       | Examples demonstrating the FastAPI endpoints for SNMP-based DOCSIS system and channel statistics. |
| [FastAPI cURL](cli-fastapi-curl.md)       | Examples demonstrating how to use cURL to interact with the FastAPI endpoints.                    |
| [Python Parser](cli-pnm-parser-py.md)     | Examples demonstrating the Python API for PNM Signal Analysis and reporting.                      |
